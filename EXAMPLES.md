# 🎮 Geometry Dash MCP Server - Примеры команд

Готовые команды для копирования в Claude Desktop после установки MCP сервера.

## 🌟 Базовые примеры

### Первый уровень

```
Создай уровень "Hello World" со сложностью easy
Добавь блок на (0, 0)
Добавь блок на (1, 0)
Добавь блок на (2, 0)
Экспортируй уровень
```

### Простая платформа

```
Создай уровень "Platform Test"

Добавь платформу на (5, 0)

```

```
Добавь блок на (0, 0)
Сделай hot reload

Добавь блок на (1, 0)
Сделай hot reload

Добавь шип на (2, 0)
Сделай hot reload

Добавь орб на (3, 3)
Сделай hot reload
```

## 📋 Управление уровнями

### Просмотр всех уровней

```
Покажи все уровни
```

### Экспорт конкретного уровня

```
Экспортируй уровень "My Level"
```

## 💡 Советы

1. **Координаты:** 1 единица = 30 пикселей в GD
2. **Сложность:** auto < easy < normal < hard < harder < insane < demon
3. **Объекты:** Экспериментируйте с разными типами!
4. **Hot reload:** Требует настройки DLL инжектора

## 🎯 Создание своих уровней

Используйте эти шаблоны как основу:

```
Создай уровень "[ИМЯ]" со сложностью [СЛОЖНОСТЬ]

[Добавьте объекты...]

Экспортируй уровень
```

Или с hot-reload:

```
Создай уровень "[ИМЯ]"
[Добавьте объекты...]
Сделай hot reload
```

## 🔧 Отладка

Если что-то не работает:

```
Покажи все уровни
```

Проверьте, создан ли уровень перед добавлением объектов.

**Совет:** Сохраняйте удачные команды для повторного использования!
