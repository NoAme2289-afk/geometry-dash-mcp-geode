## Советы по использованию

### Для создания проходимых уровней:
1. **Блоки (ID 1)** - основа платформ, ставить на Y=105
2. **Шипы (ID 8)** - препятствия, ставить на Y=135
3. **Орбы (ID 36, 84, 141)** - для прыжков, ставить на Y=200-250
4. **Порталы** - для смены режима игры

### Для красивого дизайна:
1. **Декорации (ID 211-215)** - добавляют детали
2. **Glow объекты (ID 1705-1707)** - добавляют свечение
3. **Цветовые каналы (0-999)** - для раскраски
4. **Триггеры (Pulse, Color)** - для анимации

### Координаты:
- **Ground level**: Y = 105
- **Шипы**: Y = 135 (на 30 единиц выше блоков)
- **Орбы**: Y = 200-250
- **Платформы в воздухе**: Y = 250-400

## Примечания

- Всего в Geometry Dash более 3000+ различных объектов
- ID объектов постоянно добавляются с новыми обновлениями
- Некоторые ID зарезервированы для будущих объектов
- Декоративные объекты имеют ID от 1 до 3000+
- Триггеры обычно имеют ID выше 899

## Источники для дополнительной информации

## Как добавить новые ID

Если вы нашли новые ID объектов:
1. Откройте Geometry Dash Editor
2. Найдите объект в меню
3. Создайте его в уровне
4. Используйте инструменты для определения ID
5. Добавьте в этот файл с описанием
